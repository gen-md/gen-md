## Package
- Name: @gen-md/cli
- Version: 0.1.0
- License: MIT
- Binary: `gen-md`
- Dependencies: @gen-md/core, commander

## Global Flags
All commands support these flags for conversational AI integration:
- `--prompt` - Output as conversational prompt (for agent consumption)
- `--git` - Include git context (branch, recent commits)

## Commands

### cascade
```bash
gen-md cascade <file> [--stop-at <dir>] [--max-depth <n>] [--json] [--show-merged] [--prompt] [--git]
```
Preview cascade chain for a .gen.md file.

### validate
```bash
gen-md validate <files...> [--no-check-output] [--no-check-context] [--no-check-skills] [--json] [--prompt] [--git]
```
Validate .gen.md files and check dependencies.

### compact
```bash
gen-md compact <files...> [-o <file>] [--array-merge <strategy>] [--body-merge <strategy>] [--resolve-paths] [--dry-run] [--prompt] [--git]
```
Merge multiple .gen.md files into one.

### prompt
```bash
gen-md prompt <file> [-o <file>] [--no-current-code] [--user-prompt <text>] [--examples-merge <strategy>] [--git] [--from-pr <n>] [--max-pr-examples <n>] [--json]
```
Generate a conversational prompt from a .gen.md file with one-shot examples.
- `--git` - Include git context
- `--from-pr <n>` - Include PR #n as multishot example
- `--max-pr-examples <n>` - Max number of PR examples to include

### pr
```bash
gen-md pr <file> [--title <title>] [--base <branch>] [--from-pr <n>] [--max-pr-examples <n>] [--create] [--json]
```
Generate PR-ready output from a .gen.md file.
- `--title <title>` - Override PR title
- `--base <branch>` - Target branch (default: main)
- `--from-pr <n>` - Include PR #n as multishot example
- `--create` - Create the PR using gh CLI
- `--json` - Output as JSON

## GitHub Authentication
For PR integration, auth is resolved in order:
1. `GITHUB_TOKEN` environment variable
2. `GH_TOKEN` environment variable
3. `gh auth token` command (if gh CLI is installed)

## Exit Codes
- 0: Success
- 1: Error
- 2: Validation failure
