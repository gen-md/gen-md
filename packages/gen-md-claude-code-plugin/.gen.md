## Package Type: Claude Code Plugin

This is a Claude Code plugin that provides .gen.md commands and hooks.

### Structure Requirements
```
gen-md-claude-code-plugin/
├── plugin.json            # Plugin manifest
├── commands/
│   ├── gen-md.md         # /gen-md command
│   ├── validate.md       # /gen-md-validate command
│   └── cascade.md        # /gen-md-cascade command
├── hooks/
│   └── detect-gen-md.md  # Auto-detect .gen.md files
├── skills/               # Optional skill references
└── README.md
```

### Plugin Manifest (plugin.json)
```json
{
  "name": "gen-md",
  "version": "0.1.0",
  "description": "...",
  "author": "gen-md",
  "commands": ["commands/*.md"],
  "hooks": ["hooks/*.md"],
  "skills": ["path/to/skill.md"]
}
```

### Command Format (.md files)
```markdown
---
name: command-name
description: What the command does
arguments:
  - name: arg1
    description: Argument description
    required: false
---
<input>
Instructions for Claude when this command is invoked.
</input>
```

### Hook Format
```markdown
---
event: file_open
pattern: "**/*.gen.md"
---
<input>
Instructions for Claude when a .gen.md file is opened.
</input>
```

### Commands to Implement
- /gen-md [file] - Generate output from .gen.md
- /gen-md-validate [files...] - Validate .gen.md files
- /gen-md-cascade [file] - Show cascade chain
</input>
