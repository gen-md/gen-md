## Package Type: VS Code Extension

This is a VS Code extension that provides .gen.md file support.

### Structure Requirements
```
gen-md-vscode/
├── src/
│   ├── extension.ts       # Extension entry point (activate/deactivate)
│   ├── commands/          # Command implementations
│   │   └── index.ts
│   └── providers/         # VS Code providers
│       ├── validation.ts  # DiagnosticCollection provider
│       └── hover.ts       # HoverProvider
├── syntaxes/
│   └── gen-md.tmLanguage.json  # TextMate grammar
├── language-configuration.json
├── package.json           # VS Code extension manifest
├── tsconfig.json
└── README.md
```

### VS Code Manifest (package.json)
Required fields:
- engines.vscode: "^1.85.0"
- activationEvents: onLanguage:gen-md, workspaceContains:**/*.gen.md
- contributes: languages, grammars, commands, menus, configuration

### Code Conventions
- Import @gen-md/core for core functionality
- Register commands in extension.ts activate()
- Use vscode.window for UI (showInformationMessage, createOutputChannel)
- Use vscode.workspace for file operations
- Dispose all subscriptions in deactivate()

### Features to Implement
- Commands: generate, validate, showCascade, compact
- Providers: ValidationProvider, HoverProvider
- Language: syntax highlighting, language configuration
- Settings: validateOnSave, showCascadeOnHover
